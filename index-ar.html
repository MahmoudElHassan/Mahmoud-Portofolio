مطور .NET للواجهة الخلفية

مطور .NET للواجهة الخلفية يبني واجهات REST قوية باستخدام ASP.NET Core و ‎.NET Framework، ويُدمج SQL Server و PostgreSQL و MongoDB، ويحسّن الأداء عبر Redis، وينشر خدمات قابلة للتوسّع على Azure App Service مع CI/CD. خبرة في المصادقة بـ JWT، والهندسة النظيفة، والعمل التعاوني لتسليم برمجيات عالية الجودة.

  • Numeriisoft

    قمت ببناء الواجهة الخلفية الكاملة لمنصة AstroBasic/NumeriiSoft، وهي منظومة تقدّم رؤى رقمية/كَرْمية مخصّصة ومحتوى تعليمي في علم الفلك. تم تطوير الواجهة الخلفية باستخدام ‎ASP.NET Core Web API وفق هندسة نظيفة متعددة الطبقات، مع ‎EF Core وقاعدة بيانات SQL واعتماد نمط المستودع لتسهيل الصيانة.

    يشمل النظام تكامل عدة واجهات برمجية خارجية للحسابات الرقمية اعتمادًا على بيانات المستخدم (الاسم، تاريخ الميلاد، تفاصيل الميلاد)، ثم دمج النتائج وإرجاعها كاستجابات JSON موحّدة للواجهة الأمامية.

    طوّرت أيضًا وحدة إدارة الدورات التعليمية: فهرس الدورات، التسجيلات، والتوصيات المخصّصة. تم تأمين النظام عبر مصادقة JWT، والتحقق الصارم من المدخلات، وإدارة موحّدة للأخطاء، مع توثيق كامل عبر Swagger/OpenAPI.

    زيارة Numeriisoft

  • Khzama

    طوّرت الواجهة الخلفية الكاملة لمنصّة تبرعات إلكترونية تدعم المساجد والمبادرات المجتمعية في مكة. بنيت ‎ASP.NET Core Web API بهندسة نظيفة (Domain, Application, Infrastructure, API) لضمان قابلية التطوير وسهولة الصيانة.

    حوّلت المنصّة إلى نظام SaaS متعدد المستأجرين مع فصل آمن للبيانات، ونشرتها على ‎Azure App Service مع إعدادات ‎CI/CD، والتوسّع التلقائي، والمتغيرات البيئية.

    قدّمت واجهات REST تُستهلك من بوابة الإدارة والمتجر للعملاء، مع تكامل بوابات الدفع، وتتبع التبرعات، وتوليد السجلات المالية، وحماية عبر JWT والتحقق من المدخلات.

    زيارة Khzama

  • Aqwom

    منصّة إلكترونية متكاملة لتقديم طلبات الدعم المالي للجهات وفق احتياجاتها التشغيلية، تضم بوابة للمتقدّمين ولوحة تحكّم للإدارة للمراجعة والاعتماد.

    الخلفية مبنية باستخدام ‎ASP.NET Core Web API مع هندسة نظيفة، وتقسيم واضح إلى طبقات: المجال، التطبيق، البنية التحتية، والعرض.

    تشمل الطبقات كيانات المجال وقواعد العمل، حالات الاستخدام وخرائط البيانات، وصول البيانات بـ ‎EF Core وخدمات البريد والملفات، مع واجهات API تخدم البوابة والإدارة.

    زيارة Aqwom
